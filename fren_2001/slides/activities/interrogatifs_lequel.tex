\begin{frame}{Lequel version de \lexi{lequel}?}
  Remplaçons chaque blanc avec la bonne version du pronom interrogatif \lexi{lequel}.
  \begin{enumerate}
    \item De ces deux quartiers, \underline{\uncover<2->{lequel}} est le plus chic?
    \item Ces deux filles sont de milieux différents. \underline{\uncover<3->{Laquelle}} des deux est la mieux élevée?
    \item De toutes les classes sociales, \underline{\uncover<4->{à laquelle}} est-ce que tu appartiens?
    \item Les classes ont des mœurs sociales distinctes. \underline{\uncover<5->{Auxquelles}} t'identifies-tu le plus?
    \item {}<<Les cadres supérieurs ont tous les avantages, surtout celui-là! -- \underline{\uncover<6->{Duquel}} parles-tu?>>
    \item Tu dis que tu fais partie de plusieurs équipes de sport. \underline{\uncover<7->{Desquelles}}?
    \item Parmi tous les sports, \underline{\uncover<8->{auxquels}} t'intéresses-tu le plus?
  \end{enumerate}
\end{frame}
