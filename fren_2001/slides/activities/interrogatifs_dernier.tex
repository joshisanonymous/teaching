\begin{frame}{Interrogatifs}
  Mets le bon mot interrogatif.
  \begin{enumerate}
    \item \underline{\uncover<2->{Que}} peut-on faire pour héberger les sans-logis?
    \item \underline{\uncover<3->{Avec qui}} ton oncle s'est-il marié?
    \item \underline{\uncover<4->{Quel}} couple est trop indulgent, selon toi?
    \item Parmi les deux facultés, \underline{\uncover<5->{laquelle}} préfères-tu?
    \item \underline{\uncover<6->{Qui est-ce que}} votre tante gronde?
    \item \underline{\uncover<7->{De quoi}} les doyens s'occupent-ils?
    \item \underline{\uncover<8->{Qu'est-ce qui}} a établi l'entente que les hommes et les femmes sont égaux?
  \end{enumerate}
