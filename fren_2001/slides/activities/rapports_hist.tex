\begin{frame}[b]{L'histoire d'un rapport}
  \small
  \only<1>{
    Racontons une histoire en groupes de 3 en utilisant les adjectifs et les verbes suivants.
    Pour ce faire, une personne commence par donner une phrase, puis une autre personne ajoute une deuxième phrase pour continuer l'histoire, et ainsi de suite.
    Écrivez votre histoire à mesure que vous la racontez.
    Vous pouvons commencer par cette phrase:
  \begin{itemize}
    \item Deux personnes se voient pour la première fois à ...
  \end{itemize}
  }
  \only<2->{
    Maintenant, lisez votre histoire à la classe.
    \vspace{1cm}
  }
  \begin{columns}[t]
    \scriptsize
    \column{0.28\textwidth}
      \begin{itemize}
        \item[] \alert{Adjectifs:}

        \item doux, douce
        \item exigeant.e
        \item in.fidèle
        \item jaloux, jalouse
        \item mignon.ne
      \end{itemize}
    \column{0.28\textwidth}
      \begin{itemize}
        \item séduisant.e

        \item[] \alert{Verbes:}
        \item divorce de

      \end{itemize}

      \end{itemize}
  \end{columns}
